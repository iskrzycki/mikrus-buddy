# Mikr.us Buddy

![Mikr.us Buddy Logo](path/to/logo.png)

Mikr.us Buddy is an open-source browser extension designed for the Mikr.us community. This extension leverages the Mikr.us API to display vital information about your VPS purchased from Mikr.us directly in your browser. [Install from chrome webstore](todo/link/to/extension/page)

## Features

- **VPS Information**: View detailed information about your VPS.
- **Server Management**: Restart your server with a single click.
- **Real-time Updates**: Get real-time updates on your server status.
- **User-friendly Interface**: Easy to use and navigate.

   ```sh
   git clone https://github.com/yourusername/mikr.us-buddy.git
   cd mikr.us-buddy
   ```

   ```sh
   npm install
   # or
   yarn install
   ```

   ```sh
   npm run build

   yarn build
   ```

## Installation

### Chrome

1. Download the latest release from the [Releases](https://github.com/yourusername/mikr.us-buddy/releases) page.
2. Unzip the downloaded file.
3. Open Chrome and go to `chrome://extensions/`.
4. Enable "Developer mode" by clicking the toggle switch in the top right corner.
5. Click the "Load unpacked" button and select the unzipped folder.

### Firefox

1. Download the latest release from the [Releases](https://github.com/yourusername/mikr.us-buddy/releases) page.
2. Open Firefox and go to `about:debugging`.
3. Click "This Firefox" in the sidebar.
4. Click "Load Temporary Add-on" and select the downloaded `.xpi` file.
